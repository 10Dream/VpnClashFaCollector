 تمام فایل های پروژه -  findns — اسکنر DNS Resolver برای تانل
***🔥***
نسخه v0.1.4 — تغییرات:
رفع باگ:
• رفع مشکل tunnel check (resolve/tunnel و doh/resolve/tunnel) که روی subdomain delegation همه resolverها فیل می‌شد
• رفع مشکل Ctrl+C که نتایج ذخیره نمی‌شد — الان نتایج جمع‌شده تا لحظه قطع ذخیره می‌شه
• رفع مشکل پیدا نشدن dnstt-client/slipstream-client وقتی کنار findns گذاشته بشه
• رفع خطای dnstt-client not found
قابلیت‌های جدید:
• پشتیبانی از CIDR — الان می‌تونید رنج آی‌پی مثل 
[185.51.200.0/24](http://185.51.200.0/24)
 رو مستقیم به عنوان ورودی بدید و خودکار باز می‌شه
• فلگ --proxy-auth برای پروکسی SOCKS با احراز هویت در تست‌های e2e
لینک:
[https://github.com/SamNet-dev/findns](https://github.com/SamNet-dev/findns)

### 🕒 2026-03-06 01:51:17 UTC
***🔍***
 خب اینم findns — اسکنر DNS Resolver برای تانل
ابزار findns یه اسکنر سریع و قدرتمنده که بین هزاران DNS resolver، بهترین‌ها رو برای تانل DNS (مثل DNSTT و Slipstream پیدا می‌کنه. UDP پورت 53  و هم DoH پورت 443 رو ساپورت می‌کنه. کافیه لیست resolver بگیری، اسکن بزنی و بهترین resolver رو تحویل بگیری! 
***✅***
***🔗***
 پایپلاین کامل: Ping → Resolve → NXDOMAIN → EDNS → Tunnel → E2E
***🛡️***
 تشخیص خودکار هایجک DNS
***📏***
 تست سایز EDNS برای سرعت بیشتر تانل
***🚇***
 تست واقعی end-to-end — واقعاً کلاینت DNSTT رو اجرا می‌کنه و اتصال رو تأیید می‌کنه
***⚡***
 و 50 worker موازی — هزاران resolver در چند دقیقه
***📋***
 خروجی JSON آماده استفاده مستقیم
***🌍***
 لیست داخلی 7,800+ resolver منطقه‌ای + دانلود خودکار لیست جهانی
***💻***
 پروژه: 
[github.com/SamNet-dev/findns](http://github.com/SamNet-dev/findns)
***📖***
 راهنمای کامل فارسی:
