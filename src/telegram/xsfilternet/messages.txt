7Client
یک سایت متن باز با لاراول دارم می‌سازم که کدش رو داخل گیت‌هاب کاملا رایگان قرار میدم.
این سایت چیه؟
- حساب کاربری ایجاد میکنید
- کانفیگ هاتون رو به صورت گروه بندی اضافه می‌کنید
- محدودیت دستگاه های دلخواه رو قرار میدید
- یک کاربر با محدودیت دلخواه و گروه دلخواه ایجاد میکنید
- لینک subscription را کپی میکنید و به کاربران یا کانال خود ارسال میکنید
چرا اینجوری؟
- راحت تر کانفیگ اضافه می‌کنید
- میتونید دسترسی آپدیت یک کاربر را قطع کنید
- می‌تونید تعداد دستگاه متصل به لینک را مشاهده کنید
- میتونید کنترل بهتر روی کانفیگ های عمومی خود داشته باشید
امکانات بعدی به پروژه اضافه می‌شود.
کد پروژه روی چه سرور هایی اجرا می شود؟
- کد پروژه حداقل 200 مگابایت فضا لازم دارد.
- دیتابیس MySQL حتما در سرور وجود داشته باشد (البته از تمام دیتابیس های SQL مانند SQLite و PostgreSQL نیز پشتیبانی می‌کند).
- در سرور، وب سرور LiteSpeed یا Apache یا Nginx نصب باشد.
- موقعیت سرور مهم نیست، حتی ایران.
- نوع سرور مهم نیست! 7Client بر روی تمام سرور های مجازی، سرور های اختصاصی، هاست های اشتراکی، سرویس های ابری و ... اجرا می‌شود.
- حداقل RAM مورد نیاز برای پروژه 50 مگابایت است.
- حداقل CPU مورد نیاز 0.25 است.
- پروژه به زبان php ساخته شده. پس حتما در سرور شما باید php نسخه 8.2 یا 8.3 یا 8.4 نصب باشد.
