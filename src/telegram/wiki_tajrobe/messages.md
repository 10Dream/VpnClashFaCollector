### 🕒 2026-02-10 16:49:18 UTC
**اشتراک فیلترشکن اندروید  برای ویندوز و مک (بدون نیاز به روت کردن گوشی)**
**به چه درد می خوره؟**
۱. تونل کردن کل سیستم (Full Tunnel): مشکل اصلی اینه که خیلی از برنامه‌ها و بازی‌های کامپیوتر، تنظیمات پروکسی (Socks/HTTP) ندارن و با فیلترشکن‌های معمولی ویندوز کار نمی‌کنن. با این روش، PdaNet مثل یه کارت شبکه مجازی عمل می‌کنه و کل ترافیک ویندوز رو مجبور می‌کنه از فیلترشکن گوشی رد بشه. پس همه چی (از آپدیت ویندوز گرفته تا گیم و تلگرام) بدون تنظیم خاصی وصل میشه.
2. بعضی از نرم‌افزارها قابلیت socks یا http ندارن، با این روش هر فیلترشکنی که روی گوشی متصل شده رو می تونید در ویندوز یا مک به اشتراک بزارید (مثل Psiphon)
3. سرعت و پایداری بیشتر: تجربه نشون داده کلاینت‌های موبایل (مثل v2rayNG) توی ایران خیلی بهتر، سریع‌تر و با قطعی کمتر نسبت به نسخه‌های ویندوزی (مثل v2rayN) وصل میشن. با این کار، قدرت و پایداری نسخه موبایل رو میاری روی کامپیوتر.
4. دور زدن محدودیت‌های تک‌کاربره: اگه اکانت وی‌پی‌ان پولی خریدی که فقط روی یه دستگاه کار می‌کنه (تک‌کاربره)، با این روش می‌تونی همزمان روی کامپیوتر هم ازش استفاده کنی.
5. پینگ بهتر برای گیمرها: چون اتصال با کابل USB برقرار میشه (نه وای‌فای)، نوسان پینگ (Jitter) خیلی کمتره و برای بازی آنلاین مناسب‌تره.
---
**مراحل اصلی:**
***🛠***
 مرحله ۱: آماده‌سازی گوشی (فقط یک‌بار)
۱. فعال‌سازی USB Debugging: باید به گوشی اجازه بدی اطلاعات رو با کابل رد و بدل کنه:
برو تو Settings -> Developer options -> گزینه USB debugging رو روشن کن. (
[در پست قبلی آموزش دادیم](https://t.me/wiki_tajrobe/5329)
).
۲. جلوگیری از قطع شدن (خیلی مهم!): 
***🔋***
 واسه اینکه وقتی صفحه گوشی خاموش شد، اندروید برنامه رو نبنده و اینترنت قطع نشه:
انگشتت رو نگه دار روی آیکون PdaNet+ -> بزن روی App info (علامت تعجب) -> برو تو بخش Battery.
گزینه رو بذار روی Unrestricted (یا "بدون محدودیت").
دقیقاً همین کار رو برای فیلترشکن (سایفون یا هر فیلتر شکن دیگه) هم انجام بده.
***💻***
 مرحله ۲: تنظیمات کامپیوتر (فقط یک‌بار)
نسخه ویندوز PdaNet رو دانلود و نصب کن.
بعد از نصب، اون پایین کنار ساعت ویندوز، روی آیکون PdaNet (شبیه موبایل) کلیک راست کن.
برو تو Settings و تیک گزینه Auto connect on USB plug-in رو بزن. (این یعنی کابل رو زدی، خودش وصل شه!)
***🔌***
 مرحله ۳: اتصال و استفاده (همیشگی)
فیلترشکن سایفون رو روی گوشی روشن کن.
برنامه PdaNet+ رو توی گوشی باز کن و تیک USB Tethering رو بزن.
کابل رو بزن به سیستم.
تمام! خودش اتوماتیک وصل میشه و سیستم از فیلترشکن گوشی استفاده می‌کنه.

**چطور Developer options رو فعال کنیم؟**
***📱***
 برای سامسونگ (Samsung):
تنظیمات -> About phone (درباره تلفن).
بزن روی Software information (اطلاعات نرم‌افزار).
روی Build number (شماره ساخت) ۷ بار پشت سر هم سریع ضربه بزن.
رمز گوشی رو بزن. حالا توی صفحه اصلی تنظیمات (اون پایین)، گزینه Developer options اضافه شده.
***📱***
 برای شیائومی (Xiaomi / POCO / Redmi):
تنظیمات -> About phone (درباره گوشی).
روی گزینه MIUI Version (نسخه MIUI) یا HyperOS Version حدود ۷ بار پشت سر هم سریع ضربه بزن تا بنویسه "You are now a developer".
حالا برگرد عقب، برو توی Additional Settings (تنظیمات بیشتر).
اونجا گزینه Developer options رو می‌بینی.
(برای بقیه برندها مثل هواوی و... کافیه مدل گوشی رو توی گوگل سرچ کنی).
