🛠 درباره این پروژه

پروژه VpnClashFa Collector یک سیستم مانیتورینگ و جمع‌آوری کانفیگ است که به طور مداوم منابع مختلف را اسکن کرده و پس از پایش کیفیت، آن‌ها را برای استفاده در نرم‌افزارهای مختلف بهینه‌سازی می‌کند.

اولویت‌های نمایش و محتوا:

پروکسی‌های تلگرام: لیست‌های تفکیک شده برای اندروید و ویندوز در صدر پنل.

کانفیگ‌های تست شده: بهترین کانفیگ‌ها بر اساس پینگ و سرعت دانلود واقعی (Speed Passed & Ping Passed).

میکس همه کانفیگا: تجمیع تمام منابع موجود در یک بخش واحد (اولویت سوم).

منابع اختصاصی: نمایش جداگانه هر منبع با حفظ ترتیب فایل‌های معمولی و Base64 در کنار یکدیگر.
